# 画面遷移図

最終更新: 2026-04-15（マイページの返品申請導線）

| ページID | パス | ページ名 | 認証要否 | 実装状況 |
|---------|------|---------|---------|---------|

| P-08 | /auth/signup | 新規登録 | 不要 | デザイン済み |

| P-10 | /admin/products | 商品管理 | 管理者のみ | デザイン済み |

## 遷移フロー

### 購入フロー（メイン）
```
[P-01 トップ（商品一覧）]
    │ 商品クリック
    ↓
[P-02 商品詳細]
    │ 「カートに追加」
    ↓
[P-03 カート]
    │ 「購入手続きへ」
    │ ※未ログイン → [P-07 ログイン] → 戻る
    ↓
[P-04 購入手続き]
（お客様情報 + 配送先 + 支払い方法選択）
    │ 「注文を確定する」
    ↓
[P-05 注文完了]
```

### 認証フロー
```
[P-07 ログイン]
    ├─ ログイン成功（role=ADMIN）→ [P-09 管理ダッシュボード]（/admin）
    ├─ ログイン成功（上記以外）→ callbackUrl（未指定時は /mypage）
    └─ 「新規登録はこちら」→ [P-08 新規登録]

[P-08 新規登録]
    ├─ 登録成功 → [P-07 ログイン]
    └─ 「ログインはこちら」→ [P-07 ログイン]
```

### マイページフロー
```
[P-06 マイページ]（タブ切替）
    ├─ 「注文履歴」タブ: 注文一覧 + ステータス表示 + 対象注文の「返品申請」ボタン
    └─ 「プロフィール」タブ: 名前・メール・電話・住所編集
```

### 管理者フロー
```
role=ADMIN のユーザー
    ├─ トップ・商品詳細・カート・チェックアウト・マイページ等へ直接アクセス → [P-09] へリダイレクト（proxy）
    └─ [P-09 管理ダッシュボード]
        ├─ 「商品」→ [P-10 商品管理]
        ├─ 「注文」→ [P-11 注文管理]（全会員分の参照・ステータス更新）
        └─ 「ユーザー」→ [P-12 ユーザー管理]
```

## レイアウト構成

```
RootLayout（全ページ共通）
├── Header（ロゴ・ナビ・カートアイコン・ユーザーメニュー）
├── main（各ページコンテンツ）
└── Footer（コピーライト・リンク）

AdminLayout（/admin/* 専用）
├── AdminSidebar（管理ナビ）
└── main（管理コンテンツ）
```
