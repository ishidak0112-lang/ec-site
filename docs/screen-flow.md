## 遷移フロー

### 購入フロー（メイン）
```
[P-01 トップ]
    │
    ├─ ヘッダー「商品一覧」
    │       ↓
    │  [P-02 商品一覧]
    │       │ 商品クリック
    │       ↓
    │  [P-03 商品詳細]
    │       │ 「カートに追加」
    │       ↓
    │  [P-04 カート]
    │       │ 「購入手続きへ」
    │       │ ※未ログイン → [P-10 ログイン] → 戻る
    │       ↓
    │  [P-05 購入手続き]
    │  （配送先入力 + Stripe決済）
    │       │ 決済成功
    │       ↓
    │  [P-06 注文完了]
```

### 認証フロー
```
[P-10 ログイン]
    ├─ ログイン成功 → リダイレクト先（元ページ or トップ）
    └─ 「新規登録はこちら」→ [P-11 新規登録]

[P-11 新規登録]
    ├─ 登録成功 → [P-10 ログイン]
    └─ 「ログインはこちら」→ [P-10 ログイン]
```

### マイページフロー
```
[P-07 マイページ]
    ├─ 「注文履歴」→ [P-08 注文履歴]
    │       └─ 注文クリック → [P-09 注文詳細]
    └─ 「プロフィール編集」→ [P-07 プロフィール編集]
```

### 管理者フロー
```
[P-12 管理ダッシュボード]
    ├─ 「商品管理」→ [P-13 商品一覧（管理）]
    │       ├─ 「新規登録」→ [P-14 商品登録]
    │       └─ 「編集」→ [P-15 商品編集]
    ├─ 「注文管理」→ [P-16 注文一覧（管理）]
    │       └─ 注文クリック → [P-17 注文詳細（管理）]
    └─ 「ユーザー管理」→ [P-18 ユーザー一覧]
```

## レイアウト構成

```
RootLayout（全ページ共通）
├── Header（ロゴ・ナビ・カートアイコン・ユーザーメニュー）
├── main（各ページコンテンツ）
└── Footer（コピーライト・リンク）

AdminLayout（/admin/* 専用）
├── AdminSidebar（管理ナビ）
└── main（管理コンテンツ）
```
